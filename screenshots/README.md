# Screenshots & Gameplay Media

This folder holds screenshots and GIFs used on the itch.io page.

---

## Required Assets

### Screenshots (PNG, 1280×720)

Capture these in-game at 1280×720 resolution. Use F12 or the Godot editor screenshot tool.

| Filename               | Content                                              |
|------------------------|------------------------------------------------------|
| `screenshot_01.png`    | Main menu screen                                     |
| `screenshot_02.png`    | Wave 1 combat — first enemy encounter                |
| `screenshot_03.png`    | Mid-wave combat — multiple enemy types on screen     |
| `screenshot_04.png`    | Weapon reload / weapon swap moment                   |
| `screenshot_05.png`    | Wave complete / score screen                         |

### Gameplay GIF

A short looping GIF (5–10 seconds) showing active combat.

| Filename               | Content                                              |
|------------------------|------------------------------------------------------|
| `gameplay.gif`         | Twin-stick movement + shooting at enemies            |

Recommended capture tool: [LICEcap](https://www.cockos.com/licecap/) (Windows/macOS) or [Peek](https://github.com/phw/peek) (Linux).

---

## itch.io Upload

1. Go to your itch.io game page → **Edit game**
2. Under **Screenshots**, upload all PNG files
3. Set the first screenshot (`screenshot_01.png`) as the cover image
4. Under **Embed options**, enable the **GIF** as the animated preview if supported

---

## Tips

- Use the in-game screenshot shortcut or the Godot editor's **Scene → Take Screenshot** option.
- Crop out any editor UI — export and run the game to capture clean screenshots.
- Aim for action shots with visible enemies, muzzle flash, or hit effects.
- The cover image should be visually striking — preferably a combat shot.
